AI Daily Planner is an intelligent task management web application that uses AI/ML algorithms to help users plan and schedule their daily tasks efficiently.

Key Highlights:
Natural Language Input - Add tasks by typing naturally: "Finish report by 5pm tomorrow"
Smart Scheduling - 5 algorithms (Greedy, EDF, Knapsack) optimize your daily schedule
AI Recommendations - Suggests what task to do next based on time, energy, and context
Notifications - Schedule tasks for specific times with reminder alerts
Duration Prediction - Learns from history how long tasks actually take
Tech Stack:
Frontend: Next.js 15, React, TypeScript
Backend: Node.js, Express.js
Database: SQL Server Express
NLP: Chrono.js, Compromise.js
Short Description (for GitHub):
An AI-powered daily planner with natural language task input, intelligent scheduling algorithms, and personalized recommendations. Built with Next.js and Node.js.
